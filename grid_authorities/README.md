# grid_authorities

Status: authority-comparison and architectural-study area. This folder is for research notes, external-authority mapping, and repository architecture studies. It is not the purchase-order ledger, not the quotation queue, and not the binding doctrine folder.

Purpose: research who else is doing authoritative grid and energy-data work, especially open source, and compare that ecosystem with the GlobalGrid2050 mission.
